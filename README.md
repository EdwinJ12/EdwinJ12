## 🖥️ Projects

### 🤖 All-Wheel Independent Steering System — 6-Wheeled Autonomous Rover
My engineering capstone and biggest project. Designed and built a full rover from scratch — chassis, suspension, steering algorithm, and wireless firmware.
- Custom **Ackermann steering geometry** derived via regression analysis
- **4 steering modes**: standard Ackermann, 4-wheel steer, crab steer, pivot turn
- **Rocker-Bogie suspension** with 13–16 cm articulation travel
- FEA-validated aluminium 7075 chassis with **factor of safety of 4**
- Dual **ESP32** microcontrollers, 60 m wireless range
- Physically fabricated the entire rover — chassis cutting, drilling, assembly

### ⚙️ RVDT Stator Winding Machine (Special Purpose Machine)
A 3-axis automated winding machine I designed and built at work for manufacturing RVDT stator coils used in aerospace flight control systems.
- **3-axis motion**: 2 linear + 1 rotary, driven by stepper motors
- **87% reduction** in cycle time (2 hours → 10–15 min per stator)
- **97% first-pass quality rate**
- Arduino-based closed-loop control
